A Bedrock skin pack editor built for real project files

SkinApex helps you work with Minecraft Bedrock skin packs in the browser. Open existing packs, inspect files, preview models in 3D and UV, edit bones, test bound animations and preview actions, update skin metadata, and manage animation mappings without setting up a backend.

  • Supports .mcpack, .zip, loose files, and folders
  • Includes live 3D preview, UV view, bone outline tools, and transform editing
  • Supports skin animation mapping, preview actions, and official Bedrock animation autocomplete
  • Works on desktop and mobile layouts

If you build, inspect, or maintain Bedrock skin packs, SkinApex is meant to be a focused in-browser editor instead of a generic archive viewer.

Core Features

  • Open Bedrock skin packs from archives or folders
  • Inspect `manifest.json`, `skins.json`, textures, and geometry files
  • Preview models with live 3D and UV rendering
  • Edit bones, pivots, transforms, and skin metadata
  • Manage animation mappings with official animation suggestions

Best For

  • Minecraft Bedrock skin pack creators
  • Artists reviewing geometry and textures together
  • Pack maintainers updating metadata and animation mappings
  • Developers who want a lightweight browser-based editing workflow
